# Web Scraping with Puppeteer and Nodejs

### install
```
npm init -y
npm i puppeteer
```
### add to package.json
"type": "module"

### run
```
node scraper
```
### create .env file
```
SCRAPER_URL=
SEARCH_TERM=
```
### test
```
npm run test
```
